# 🎬 The Film Vault

> Discover, explore, and fall in love with movies — your ultimate film companion.

A clean and intuitive movie discovery platform. Browse popular films, search by title or genre, and get detailed information including ratings, release dates, and descriptions. Built with love for cinema and code.

## ✨ Features

- 🔍 Search Movies — find any film by title or keyword
- 🎞️ Movie Details — get ratings, overview, release date, and genres
- 🖼️ Poster Previews — see high-quality movie posters instantly
- 📱 Responsive Design — works perfectly on mobile, tablet, and desktop
- ⚡ Fast & Lightweight — optimized for speed and smooth UX
- 🌙 Clean UI — minimalist interface focused on what matters: the films
